## 🚀 快速开始

### 1. 安装与配置
```bash
# 进入项目目录
cd /path/to/yyc3-admin-system-2

# 进入诊断工具目录
cd problems_and_diagnostics

# 安装依赖并配置环境
./setup-diagnostics.sh
```

### 2. 基础使用
```bash
# 系统健康检查
./diagnose.sh --health-check

# 快速检查
./diagnose.sh --quick

# 性能分析
./diagnose.sh --performance

# 安全扫描
./diagnose.sh --security
```

### 3. 全局命令
```bash
# 配置全局命令后使用
diagnose --help
diagnose --health-check
```

## 💡 最佳实践

### ✅ 日常操作
1. **定时健康检查**
   ```bash
   # 添加到crontab
   0 */6 * * * /path/to/diagnose.sh --health-check
   ```

2. **性能基准建立**
   ```bash
   # 建立初始性能基线
   ./diagnose.sh --performance --baseline
   ```

3. **定期安全扫描**
   ```bash
   # 每周安全扫描
   0 2 * * 0 /path/to/diagnose.sh --security
   ```

### 🎯 告警设置
- **CPU使用率**: >80%警告，>90%严重
- **内存使用率**: >85%警告，>95%严重
- **磁盘使用率**: >85%警告，>95%严重
- **响应时间**: >500ms警告，>1000ms严重

### 📊 报告管理
- **日报告**: 保留30天
- **周报告**: 保留12周
- **月报告**: 保留12个月
- **年度报告**: 长期保留

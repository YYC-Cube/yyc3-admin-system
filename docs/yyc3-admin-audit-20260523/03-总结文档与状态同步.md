## 关键决策记录

| 决策 | 背景 | 选择 | 原因 |
| --- | --- | --- | --- |
| 使用 pnpm 替代 npm | 项目有 pnpm 但实际用 npm | 迁移到 pnpm | 更好的磁盘效率、严格依赖隔离、workspace 支持 |
| 使用 jose 替代 jsonwebtoken | JWT Token 需 Edge Runtime 兼容 | jose 库 | Edge Runtime 原生支持 Web Crypto API |
| 保留 Mock 服务作为开发降级 | 生产需数据库认证 | 双模式：开发可 Mock，生产必须 DB | 平滑过渡，不破坏开发体验 |
| 分阶段执行 | 大量改进项需有序推进 | 5 阶段按优先级排序 | 安全第一，依赖其次，架构第三 |

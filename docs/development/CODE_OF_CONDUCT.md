# 行为准则 📜

## 我们的承诺

为了营造一个开放和包容的环境，我们作为贡献者和维护者承诺，无论年龄、体型、残疾、种族、性别认同和表达、经验水平、国籍、个人外貌、宗教或性取向如何，参与我们的项目和社区都将不会遇到骚扰。

## 我们的标准

有助于创造积极环境的行为示例包括：

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 专注于对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用性化语言或图像，以及不受欢迎的性注意力或进步
- 讽刺、侮辱性/贬损性评论，以及个人或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息，如物理或电子邮件地址
- 在专业环境中可能合理地被认为不适当的其他行为

## 我们的责任

项目维护者负责明确可接受行为的标准，并应采取适当和公平的纠正措施回应任何不适当行为的实例。

项目维护者有权和责任删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、wiki编辑、问题和其他贡献，或者暂时或永久禁止任何贡献者从事其他他们认为不合适、威胁性、冒犯性或有害的行为。

## 适用范围

本行为准则适用于项目空间内的所有活动，也适用于个人在公共空间代表项目或其社区时的行为。代表项目或社区的例子包括使用官方项目电子邮件地址、通过官方社交媒体账户发帖或在在线或离线活动中担任指定代表。

## 执行

滥用、骚扰或其他不可接受行为的实例可向负责执行的项目团队报告，联系方式为 opensource@yyc3.dev。所有投诉将被审查和调查，并将根据情况的严重性采取适当的回应。项目团队有义务对报告事件的记者保密。

### 执行流程

1. **报告接收**：收到投诉后，项目维护者将在 48 小时内确认收到
2. **初步评估**：维护者将评估投诉的严重性和可信度
3. **调查**：可能需要联系相关方获取更多信息
4. **决策**：根据调查结果，维护者将决定采取的措施
5. **通知**：通知投诉人和被投诉人决策结果和后续步骤
6. **跟进**：确保所有措施得到执行，并监控情况

### 可能的后果

根据行为的严重性和频率，违规者可能面临以下后果：

- 私下警告
- 公开警告
- 临时禁止参与项目活动
- 永久禁止参与项目活动
- 报告给相关平台或机构（如涉及严重违规）

未能真诚遵守或执行行为准则的项目维护者可能面临由项目领导层其他成员决定的临时或永久后果。

## 升级指南

### 从v3.0升级到v4.0

1. **备份数据**
   \`\`\`bash
   # 备份数据库
   mysqldump -u root -p yyc3_yy > backup.sql
   \`\`\`

2. **更新代码**
   \`\`\`bash
   git pull origin main
   npm install
   \`\`\`

3. **运行迁移**
   \`\`\`bash
   npm run migrate
   \`\`\`

4. **配置环境变量**
   \`\`\`bash
   cp .env.example .env.local
   # 编辑.env.local配置
   \`\`\`

5. **启动服务**
   \`\`\`bash
   npm run build
   npm run start
   \`\`\`
